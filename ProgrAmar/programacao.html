LINGUAGENS BÁSICAS DA PROGRAMAÇÃO WEB

PARA QUE SERVEM?

HTML

HTML (Hypertext Markup Language) é a linguagem de marcação padrão para criar páginas da web. Ele define a estrutura e o conteúdo de uma página web usando uma série de elementos e tags. Os elementos HTML consistem em tags que envolvem o conteúdo para descrever seu significado e propósito na página.

CSS

CSS (Cascading Style Sheets) é uma linguagem de programação usada para estilizar páginas da web. Basicamente, é como se fosse o "maquiador" do site, controlando as cores, fontes, layouts e outros aspectos visuais para tornar as páginas bonitas e fáceis de usar.

JavaScript

JavaScript é uma linguagem de programação amplamente utilizada para criar e controlar conteúdo dinâmico em páginas da web. Com o JavaScript podemos adicionar interatividade ao site, criar efeitos visuais, animações, alterar os conteúdos de forma mais simples e dinâmica.

EVOLUÇÃO DAS LINGUAGENS

A evolução das linguagens de programação web tem sido marcada por avanços tecnológicos significativos, que transformaram a maneira como desenvolvemos e interagimos com a web. Abaixo está um resumo da evolução das principais linguagens e tecnologias de programação web:

LINHA DO TEMPO

  • Anos 90: Criação e estruturação básica da web com HTML, CSS e o início da interatividade com JavaScript.
  • Anos 2000: Expansão da web dinâmica com PHP, Ruby on Rails e o advento do AJAX.
  • Anos 2010: Avanço dos frameworks JavaScript (jQuery, Angular, React, Vue.js), Node.js, HTML5 e CSS3.
  • Anos 2020: Evolução para PWAs, JAMstack, e melhorias em ferramentas de desenvolvimento e linguagens como TypeScript.
